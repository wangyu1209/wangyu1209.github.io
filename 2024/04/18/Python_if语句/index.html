if-elif-else结构

语法示例：

  • if语句用于检查一个条件是否为真，如果为真，则执行相应的代码块。
  • elif语句用于检查另一个条件是否为真，如果前面的条件都不为真且该条件为真，则执行相应的代码块。
  • else语句用于处理所有条件都不为真的情况，执行相应的代码块。
